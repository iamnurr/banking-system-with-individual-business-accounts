
## About Banking Transaction System

### Installation

After clone go to file repo open terminal run commands one by one

```php
composer install
```

```php
cp .env.example .env
```

```php
php artisan key:generate
```

#### setup .env file
```php
php artisan migrate
```

```php
npm install
```

```php
npm run dev
```

```php
php artisan serve
```

project will be live at http://127.0.0.1:8000
